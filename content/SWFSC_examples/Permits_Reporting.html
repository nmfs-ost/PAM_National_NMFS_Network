Permit Reporting

An automated script can be found here for pulling the necessary protected resources reporting data from our sightings data. This script does not produce the final Take Report needed for this permit. It does return takes per species, which can be added to the final Take Report provided by the permit coordinator. Download the script and follow the directions for each reporting cycle. Contact Kourtney Burger (kourtney.burger@noaa.gov) with any questions/concerns.
